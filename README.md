# Monitoring Suhu dan Kelembaban Ruangan via ThingSpeak

## Deskripsi
Proyek Akhir Sistem Mikrokontroler berupa sistem monitoring suhu dan kelembaban ruangan berbasis Internet of Things (IoT).

## Komponen
- ESP32
- Sensor DHT22
- LCD I2C 16x2
- LED Hijau
- LED Kuning
- LED Merah
- Buzzer
- Resistor

## Software
- Wokwi Simulator
- ThingSpeak

## Protokol Komunikasi
- HTTP (Hypertext Transfer Protocol)

## Fitur
- Monitoring suhu dan kelembaban secara real-time.
- Menampilkan data pada LCD.
- Mengirim data ke ThingSpeak.
- Menampilkan indikator kondisi ruangan menggunakan LED dan buzzer.
